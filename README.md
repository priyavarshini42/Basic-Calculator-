# Calculator Application

A responsive calculator application developed using **HTML, CSS, and JavaScript**. It features a clean and user-friendly interface with fully functional arithmetic operations, including addition, subtraction, multiplication, division, clear, and backspace functionality.

## Features

* Responsive design
* User-friendly interface
* Basic arithmetic operations:

  * Addition (+)
  * Subtraction (-)
  * Multiplication (×)
  * Division (/)
* Clear button (C)
* Backspace functionality (⌫)
* Decimal number support
* Interactive button functionality using JavaScript

## Technologies Used

* **HTML5** – Structure of the calculator
* **CSS3** – Styling and layout
* **JavaScript** – Calculator functionality and operations
